# pcb_design
These are my pcb design
This project is a system that has functions like these: 2 channel wave generator, 2 channel frequency counter,  and 2 channel adjustable power supply (18V-0V).
Using this project should follow this process: 1. Write and program the MCU. 2. Use type-C to connect to power supply. 3. Use the rotating switch to control the display on the screen and turn on whatever function needed or turn off. 4. Connect the output (pin headers) to the devices that the user want to receive the output or connect the input (for frequency counter) to count the frequency.
I made this project because I only have a 1-channel wave generator, 1-channel frequency counter, and no adjustable power supply at home. Then, I wanted to build a better system with 2-channel of those 3 functions needed.
